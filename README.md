Тренажер для изучения английского языка

 О проекте

Веб-приложение для изучения иностранных слов методом флеш-карточек. Позволяет создавать карточки со словами, просматривать их и проверять себя в интерактивном квизе.

Проблема, которую решает проект: Традиционное заучивание слов по бумажным карточкам неудобно и неэффективно. Приложение цифровизирует этот процесс, добавляет автоматическую проверку знаний и делает обучение более интерактивным.

 Функциональность

-  Добавление новых слов с переводом
-  Просмотр всех слов в виде карточек
-  Удаление слов
-  Интерактивный квиз для самопроверки
-  Сортировка слов по алфавиту
-  Счетчик количества слов

 Технологии

- Python 3.x
- Django 5.x
- SQLite
- HTML/CSS
